| Abbreviation | Latin | Meaning | When to use |
|---|---|---|---|
| `e.g.,` | *exempli gratia* | for example | non-exhaustive example — "e.g., RK4" |
| `i.e.,` | *id est* | that is / in other words | full clarification or restatement |
| `cf.` | *confer* | compare / see also | related source that doesn't directly support but is relevant |
| `viz.` | *videlicet* | namely | exhaustive list — more formal than i.e. |
| `n.b.` | *nota bene* | note well | flag something important, common in footnotes |
| `vs.` | *versus* | against / compared to | comparison |
| `et al.` | *et alii* | and others | 3+ authors — BibLaTeX handles automatically |
| `p.` / `pp.` | *pagina / paginae* | page / pages | citation postnotes — BibLaTeX usually adds automatically |
| `ibid.` | *ibidem* | in the same place | same source as previous citation — BibLaTeX handles automatically |
| `passim` | *passim* | throughout | idea scattered across a work, no single page |
| `etc.` | *et cetera* | and the rest | ⚠ avoid in formal writing — restructure or use "and so on" |
| `op. cit.` | *opere citato* | in the work cited | ⚠ obsolete — BibLaTeX handles this automatically |

The two rules worth memorising: **e.g. ≠ i.e.** and **always put a comma after both** in American academic English.
